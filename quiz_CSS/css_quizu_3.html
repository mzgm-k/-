CSSのbox-shadowプロパティの値の指定方法として間違っているものを選んでください。





:box-shadow: 1px;
box-shadowプロパティの値の数値は、2個・3個・4個のいずれかを
連続して指定する必要があります。
よって、数値が1つしか指定されていないAは間違いななので正解です
box-shadow: 1px 1px;
不正解です
box-shadow: 3px 3px 6px 2px;
不正解です
box-shadow: 3px 3px 6px 2px inset;
不正解です
