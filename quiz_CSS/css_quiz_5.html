CSSのbackground-sizeプロパティの値として
指定できないキーワードを選択してください





auto
プロパティの値の初期値はautoになります
fill
fillは、background-sizeプロパティには指定できません
cover
coverは単体でのみ指定可能な値で、画像を
「背景画像単体で表示領域全体を覆う最小サイズ」にします
contain
containも単体でのみ指定可能な値です
このキーワードを指定すると
「画像の全体が表示される最大サイズ」で表示されます
