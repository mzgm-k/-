次のCSSセレクタの中で、優先順位がもっとも低くなる指定を１つ選んでください。





:hover:first-child
:hover:first-childは擬似クラス2つで構成されているので優先順位は20です。
div:hover:first-child
div:hover:first-childは擬似クラスが2つと、それらの前に要素が1つありますので
優先順位は21となります。
*:hover::first-letter
*:hover::first-letterの * はカウントされず、擬似クラスが1つと擬似要素が1つですので優先順位は11です。
div:hover::first-letter
div:hover::first-letterは要素が1つと擬似クラスが1つと擬似要素が1つですので
優先順位は12となります。
