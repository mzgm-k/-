次のJavaScriptコードを実行するとどうなるのか選択してください

JavaScript
let element = document.getElementById('button');
element.textContent = 'クリックして下さい';




buttonクラスの全てのボタンのテキストを「クリックして下さい」に変更する
実装するためのコードはこちらです
JavaScript
const buttons = document.querySelectorAll('.button');
buttons.forEach(function(button) {
  button.textContent = 'クリックして下さい';
});
idがbuttonの要素のテキストを「クリックして下さい」に変更する
正解です
