指定された回数だけ繰り返し処理を行うための制御構造を選択してください





if文
条件が満たされたときに特定のコードを実行するためのものです
条件が満たされない場合、別のコードを実行することもできます
switch文
条件に基づいて異なる処理を実行するための制御構造のことです
for文
指定された回数だけ繰り返し処理を行うための制御構造のことです
while文
条件がtrue(真)の間、特定のコードブロックを繰り返し実行するループ構造のことです
